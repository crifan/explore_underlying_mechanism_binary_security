# 可执行文件

## 常见可执行文件格式

* 不同系统
  * `Windows`
    * `PE`
      * =`Portable Executable`
    * `COFF`
        * =`Common Object File Format`
  * `Linux`类系统：`Linux`、`Unix`、`Mac`等
    * `ELF`
        * =`Executable and Linkable Format`
  * `Mac`
    * `Mach-O`
* 其他相关
  * 二进制往往还根据系统不同而略有不同
    * 举例
      * Windows系统
        * `32位`=`x86`
        * `64位`=`x64`
